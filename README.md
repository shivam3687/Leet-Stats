
Leet_Stats

A simple web application to display your LeetCode statistics.

Features:

Enter a LeetCode username to fetch and display their stats.

View a breakdown of solved problems by difficulty (Easy, Medium, Hard) using progress circles.

See the total number of submissions for each difficulty.

The application uses a responsive design with a dark theme, linear-gradient background, and vibrant colors.

The username is validated using a regular expression.

How to Use:

Simply open the index.html file in your web browser. Enter a valid LeetCode username in the input field and click the "Search" button to view the statistics. The application fetches the data from the https://leetcode-stats-api.herokuapp.com/ API.

Technologies Used:
